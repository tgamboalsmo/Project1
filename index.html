Project P1: Build a Portfolio Site

The Udacity student was provided with a design mockup as a PDF-file and was expected to replicate that design in HTML and CSS. The student was to develop a responsive website that would display images, descriptions and links to each of the portfolio projects the student completed throughout the course of the Front-End Web Developer Nanodegree.

Project P2: Interactive Resume

In this project, the Udacity student demonstrated their mastery of the language's syntax through a series of challenges. Each multipart problem mimics a real-life challenge that front-end developers face. The student was required to write clean code and to apply their knowledge of variables, objects, JSON, functions and control flow to successfully solve the challenges.

Project P3: Classic Arcade Game Clone

In this project, the student was provided with visual assets and a game loop engine; using these tools the student added a number of entities to the game including the player characters and enemies to recreate the classic arcade game Frogger.

Project P4: Website Optimization

The student optimized a provided website with a number of optimization- and performance-related issues so that it achieves a target PageSpeed score and runs at 60 frames per second.

Project P5: Neighborhood Map Project

In this project, the student developed a single-page application featuring a map of their neighborhood or a neighborhood the would like to visit. They added additional functionality to this application, including: map markers to identify popular locations or places they'd like to visit, a search function to easily discover these locations, and a listview to support simple browsing of all locations. The student then researched and implemented third-party APIs that provide additional information about each of these locations (such as StreetView images, Wikipedia articles, Yelp reviews, etc).

P6: Feed Reader Testing

In this project, the student learned about testing with Javascript. Testing is an important part of the development process and many organizations practice a standard known as "test-driven development" or TDD. This is when developers write tests first, before they ever start developing their application.

Whether you work in an organization that writes tests extensively to inform product development or one that uses tests to encourage iteration, testing has become an essential skill in modern web development!
